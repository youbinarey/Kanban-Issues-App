
# Kanban Issues App

Aplicación desarrollada en **Flutter** para la gestión visual de incidencias (issues) siguiendo un estilo **Kanban**.  
Permite visualizar, clasificar y gestionar issues por estado en un tablero dividido en columnas como **Backlog**, **In Progress** y **Done**.

---

## Vista de funcionalidades
Cambio de estado de una issue.

---

##  Características

- **Tablero Kanban** con clasificación por estado.
- **Tarjetas de issue** con:
  - Código de issue.
  - Descripción breve.
  - Botón de acceso a detalles.
- **Vista de detalles** con descripción ampliada y acciones.
